# Personae (Анализа **длинных** тексов)

**_я люблю книжки с картинками_**

## Задача

- импортировать исходный файл (fb2, epub, html)
- выделить всех персонажей (людей)
- сгруппировать разные способы упоминая одного человека в одном персонаже
- сформировать информацию для поиска упоминания в исходном тексте
- сформулировать краткое описание для каждого персонажа по тексту
- понять и сформулировать тип отношений между персонажами, если они есть
- если персонаж является реальным историческим лицом, найти ссылку на страницу в интернете посвященную этому персонажу
- найти фотографию
- попробовать сделать иллюстрацию
- результат анализа сохранить в виде файла
- экспортировать html c описанием персонажей и картинками, синхронизированными с текстом
